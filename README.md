gofortune
=========

A Go application that tells fortune based on your memorized card.<br>

How to build and run
====================
```
$ git clone https://github.com/gophergala2016/gofortune
$ cd gofortune
$ go build
$ ./gofortune
```
Navigate with a browser to: ```http://localhost:8080```
